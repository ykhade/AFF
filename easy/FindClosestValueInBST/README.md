# Find the closest element to a target value in a BST

Given a non-empty search tree and a target value, find the value in the Binary Search Tree that is closest to the taret.

````
Input
-----
Binary Search Tree:
     9
   /   \
  4      17
 / \      \
3   6      22
   / \     /
  5   7   20

target: 18 


Output
------
17
